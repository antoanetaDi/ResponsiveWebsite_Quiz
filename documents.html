Dokumentation

Die ganze Dokumentation des Projektes steht auf GitHub zur Verfügung.

Kurzbeschreibung:

Das ist eine Web-Anwendung mit responsiven Elementen, Quiz mit 3 verschiedenen Themen vom Javascript-HTML Advanced Kurs bei Cimdata Bildungsakademie, Berlin. Datenspeicherung ist mittels Web-Storage-Technologien realisiert. Für die Anschrift des Kursanbieters ist Geolocation benutzt. Es gibt eine Einbindung von jQuery, Chart und Leaflet Bibliotheken.

Responsive Web Design:

Die Anzeige der Web-Seite soll sich an die Anforderungen des Endgeräts anpassen. Höchstens werden 3 unterschiedliche Größen und Bildschirmauflösungen berücksichtigt. Der Content wird basierend auf einem 12-spaltigen Grid dargestellt. Der Hauptinhalt befüllt dabei die ersten zwei Drittel des Grids, das restliche Drittel steht für ergänzende Elemente wie z.B. zusätzliche Informationen zur Verfügung. Es werden Breakpoints eingebaut, damit bei der Änderung der Seitenbreite das Menu anders dargestellt wird. Optional werden die Anzahl der Spalten geändert.

Das Quiz:

Das Quiz ist in 3 verschiedenen Themen von unserem Kurs – HTML 5, CSS 3 und jQuery. Die Fragen werden von JSON-Dateien geladen und die Resultate mit Hilfe von einem Chart gezeigt. Ich habe auch einen Timer eingesetzt. Wenn die Zeit vorbei ist, darf man das Quiz noch einmal machen. Es gibt Verbindung zum LocalHost und mit der Hilfe von JSON werden die Ergebnisse gespeichert und gezeigt. Am Ende kann man seine Resultate überprüfen und sich die richtigen Antworten anschauen.

Geolocation:

In der Contact-Seite wird Geolocation und zwar OpenStreetMap benuzt, damit die Anschriften des Kursanbieters angezeigt werden. Die cimdata-Anschriften in Berlin werden in einem GEOJSON-File gespeichert. Es wird diesbezüglich Leaflet Bibliothek verwendet. Die Adressen sind auch in einem Accordion-Menu zu sehen. Dazu wird jQuery UserInterface verwendet.

Technologien:

Um den Aufwand der reinen Darstellung gering zu halten, findet hierbei jQuery UI Verwendung. Für die funktionale Implementierung soll umfangreich und möglichst konsequent jQuery zum Einsatz kommen, ergänzt durch Standard-JavaScript.

Datenspeicherung:

Im Rahmen dieses Projekts werden die Daten mittels LocalStorage und JSON übertragen. Die Ergebnisse der Datenverarbeitung werden zur Anzeige oder Ausgabe an den lokalen Client-Rechner des Benutzers übertragen.
