KURSDETAILS

Cimdata Dozent: Klaus Domass

Inhalte:

Sie erlangen umfassende Kenntnisse in der aktuellen Webentwicklung. Ihnen werden grundlegende Konzepte für die Erstellung von Webapplikationen vermittelt. Dabei wird besonders auf das Zusammenspiel der einzelnen Websprachen, HTML5, JavaScript, CSS3, XML, JSON, AJAX Wert gelegt. Des weiteren wird die Nutzung von JavaScript-Bibliotheken und Frameworks im Allgemeinen sowie bei speziellen Lösungen in der Webentwicklung geschult. Im Einzelnen werden spezielle HTML-Elemente (Canvas, SVG) vorgestellt und ihr Einsatz an aktuellen Beispielen geschult. Weitere Themen werden Ihnen von unseren praxisorientierten Dozenten/-innen in dieser Weiterbildung Schritt für Schritt vermittelt:

Spielentwicklung mit Canvas, Spritesheets Objektorientierte Programmierung mit JS JavaScript-Bibliotheken, jQuery jQuery Erweiterungen, jQuery UI Responsive Webdesign, mobile Websites, Media Querys Geolocation, Map's mit Google API, Map's mit Open Layers API SVG Grafiken XML Steuerung von Audio und Video mit JS

Die Vertiefung der Lerninhalte werden mit Praxisbeispielen geübt. Das Bildungsangebot schließt mit einer themenbezogenen, einwöchigen Projektphase ab.

In dieser Weiterbildung werden die Möglichkeiten von HTML5 als Technologie für die Entwicklung moderner Applikationen für Web Anwendungen vermittelt. Das Ausbauen und Festigen des Wissens im professionellen Umgang der erlernten Websprache liegen im Focus des Kurses. Sie erlangen die Fähigkeit, Web-Applications mit Unterstützung von HTML5, JavaScript und CSS3 zu entwickeln. Sie lernen, mit den HTML5-Spezifikationen zu arbeiten und diese bei der Umsetzung von Webseiten zu verwenden.

Voraussetzungen:

Die Teilnehmer/-innen sollten gute Kenntnisse der Auszeichnungssprachen HTML5 und CSS3 haben. Grundkenntnisse der Programmiersprache JavaScript sind erforderlich. Der Umgang mit gängigen Editoren / Browsern wird vorausgesetzt.

Inhalte des Moduls nach Tagen aufgeschlüsselt:

Tägliche Übungs- und Trainingseinheiten.

Tag 1: Jquery Einführung. Selektoren, CSS Handling - Bilder Galerie
Tag 2: DOM Manipulation mit Jquery. Grundlagen und Einführung in die Canvas API Teil 1. Zeichnen von einfachen Strukturen: Rechteck, Kreise, Pixelgrafik Animationstechnik. Verläufe. HTML5 Advanced Modul Dozent: Klaus Domass
Tag 3: Grundlagen und Einführung in die Canvas API Teil 2 Konstruktion einer Lupe
Tag 4: Grundlagen und Einführung in die Canvas API Teil 3 Sprite Animation. Vererbung und Objektorientierte Programmierung mit JS am Beispiel von Sprite Animation. Pixelmanipulation.
Tag 5: Aufbau eines einfachen Spiel z.B. Space Invader, Memory oder ähnliches. Kollisionserkennung bei komplexen Spielsituationen.
[Ab Tag 5: fortlaufender Workshop in den Nachmittagsstunden am Beispiel der Applikation Spiel von Tag 5. Einbau verschiedener Level, Hiscorelisten. In diesem Workshop bietet sich die Möglichkeit, die nachfolgenden Themen in die Spielentwicklung einzubinden. Es bieten sich außerdem viele Möglichkeiten mit JS Objekten zu arbeiten und somit eine sehr komplexe Form der Programmierung kennenzulernen.]
Tag 6: Installation eines lokalen Webservers. Einführung in die Bereiche LocalStorage und SessonStorage. Einführung in JSON / CSV. Speichern und Lesen von JSON Strings, am Beispiel einer Highscore Liste.
Tag 7: Einführung in die Google Maps API. Verarbeitung von Geo JSON Daten
Tag 8: Einführung in Open Street Maps API . Einführung in Leaflet.js Open Layer Technik Verarbeitung von Geo JSON Daten
Tag 9: AJAX Methoden zum Auslesen externer CSV(Comma-separated values ) / JSON Daten. HTML Visualisierung dieser Daten Quellen in Listen und Tabellen Skriptlösung mit Jquery und Javascript;
Tag 10: Gestaltung eines interaktiven Quiz grafische Auswertung mit chart.js Speichern der Nutzer Ergebnisse im local Storage HTML5 Advanced Modul Dozent: Klaus Domass
Tag 11: Audio und Video Einbindung mit Javascript / HTML5, Audio API / Audio Frameworks
Tag 12: Responsive Webdesign Umgang mit Media Queries Erstellen eines Printlayouts
Tag 13: Einführung in XML und SVG Grafik. CSS für SVG Grafik. SVG Animation. Beispiel einer Analog Uhr mit SVG / Canvas.
Tag 14: Weiterführende CSS3 Techniken Gestaltungsmöglichkeiten mit CSS3: gradient, text-shadow, etc. Keyframe Animation.
Tag 15: Projektentwicklung / Projekt Konzeption Von der Idee zum Produkt Wiederholungen und Vertiefungen
Tag 16 – 20: Projektphase

In diesem Teil der Weiterbildung können die vorgestellten Möglichkeiten von HTML5 / Javascript praktisch angewendet werden. Die komplexen Techniken verfestigen sich in der praktischen Auseinandersetzung mit den verschieden Themengebieten.

Die Teilnehmer haben die Möglichkeit im Team oder einzeln eine Applikation zu entwickeln. Bei Teamarbeit (empfohlen!) ist auch die Aufgabenteilung zu bestimmen. Für die Beurteilung der Projekte ist es nicht zwingend, das ein Konzept zu 100% umgesetzt wird.
